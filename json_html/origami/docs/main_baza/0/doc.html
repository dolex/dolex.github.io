"Квадрат" базовая складка

Схема сборки:

1. Изготовление базовой складки "квадрат" начинается с квадратного листа. На схеме складка "квадрат" представлена двумя линиями долиной и одной горой.

2. Готовим первую складку долиной по горизонтали с последующим разгибанием.

3. Выполняем складку по горизонтали. Разгибаем. Готовим вертикальную складку долиной со последующим разгибанием.

4. Выполняем складку по вертикали. Разгибаем. Готовим лист к повороту на 45 градусов.

5. Поворачиваем лист на 45 градусов. Готовим горизонтальную складку горой с последующим разгибанием.

6. Выполняем горизонтальную складку горой. Разгибаем. Убедитесь, что у Вас получилось три линии сгиба.

7. Сначала надо выполнить две складки долиной, затем две складки горой.

8. Начинаем выполнение складок долиной.

9. Заканчиваем выполнение складок долиной. Готовим выполнение двух складок горой.

10. Выполняем складки горой. Ровняем. Подготовиться нажать сверху вниз.

11. Нажимаем сверху вниз.
