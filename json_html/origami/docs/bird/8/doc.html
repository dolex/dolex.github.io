Попугайчик оригами из бумаги

Как сделать попугайчика из бумаги?

Схема по сборке:

1. Наметьте линии сгибов, как на фото: сложите вдоль и поперек пополам, боковые стороны складывайте к середине, но прогладьте только до линии пересечения.

2. Сделайте еще две складки, формирующие по бокам прямоугольные треугольники.

3. Боковые стороны сложите к центру, получившиеся в середине уголки расположите вертикально.
