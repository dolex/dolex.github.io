Кораблик оригами из бумаги

Этот кораблик из бумаги выглядит необычно и оригинально. К тому же, его просто сделать: справится и ребёнок. Кораблик особенно хорошо будет смотреться, если взять для него лист бумаги цветной с одной стороны и белый с другой. Обратите внимание: этот кораблик делается из квадрата.

Схема сборки:

Сложите квадратный лист бумаги по диагонали пополам, так чтобы цветная сторона бумаги оказалась внутри, и загладьте пальцами сгиб.

Затем по линии, обозначенной пунктиром, загните верхний угол вниз и загладьте сгиб.

Снова разверните только что загнутый угол.

Теперь нужно вывернуть отогнутый внешний сгиб наружу, так чтобы большой треугольник оказался внутри внешней отогнутой детали.
