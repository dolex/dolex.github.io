Бабочка оригами из бумаги

Для создания бабочки нам достаточно будет квадратного листа цветной бумаги.

Схема сборки:

Перед изготовлением самой бабочки нам нужно сделать заготовку. На первом этапе складываем квадрат пополам в поперечном направлении.

Затем выполняем еще один поперечный сгиб. Так у нас должно получиться.

Следующие сгибы мы будем делать по диагональным линиям. Они должны быть выполнены на другую сторону, чем поперечные сгибы.

За счет сделанных сгибов заготовку будущей бабочки мы можем сложить в виде двойного треугольника.

Теперь начнем формировать крылышки. Для этого нужно выполнить сгиб верхнего слоя треугольника вверх. Сначала делаем это с правой стороны.

Затем симметричный сгиб выполняем слева.

Далее переворачиваем нашу заготовку следующим образом. Треугольная заготовка бабочки оказалась расположена широким основанием вверх.

Нижний уголок этого треугольника загибаем вверх таким образом, чтобы он немного (на 5 мм) выступал над верхним основанием. При этом нижний слой мы не трогаем, это будут маленькие крылышки будущей бабочки.

Выступающий небольшой уголок в верхней части загибаем на другую сторону. Это будет голова нашей бабочки.

Теперь нам осталось перевернуть заготовку на другую сторону и немного согнуть в поперечном направлении.

Наша бабочка из бумаги готова. При желании можно сделать несколько таких поделок в технике оригами, выполненных в различном цвете.
