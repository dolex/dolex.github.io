Морской конёк оригами из бумаги

Схема сборки оригами:

1. Сделайте складки по диагоналям листа, затем сложите стороны к центральной линии, как показано

2. Сложите стороны снизу аналогичным образом

3. Сложите все четыре стороны одновременно

4. Должна получиться такая фигура, сложите её левую сторону назад

5. Расплюсните кармашки с обоих сторон, как на следующем фото.

6. Теперь нужно сделать складки внутрь на крайних сторонах кармашков.

7. Сложите кармашки

8. Сложите правую часть налево, совместив указанные точки, проделайте этот шаг и на другой стороне модели

9. Сделайте складку по указанной линии, затем разверните

10. Теперь по складке, сделанной в 9 шаге сложите верхнюю часть модели назад.

11. Таким образом! теперь по указанной складке сложите верхнюю часть направо, тем же способом что и в 10 шаге.

12. Нижнюю часть модели складываем внутрь. Формируем голову, загибая уголки и делая аналогичные складки.

13. Почти готово! Осталось сформировать хвост, делая его закруглнным, для этого делайте складки внутрь по указанным на картинке линиям!

14. Поздравляю, морской конёк оригами готов!
